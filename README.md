A modern Apple-inspired e-commerce website built with React and Tailwind CSS, delivering a premium shopping experience with a clean interface, responsive design, and powerful management features.

## ✨ Features

### 🛍️ Shopping Experience
- Apple-inspired modern and minimalist UI
- Fully responsive design
- Dark & Light mode
- Product search
- Product filtering
- Dynamic product pages
- Color selection
- Storage selection with automatic price updates
- Product recommendations
- Shopping cart
- Checkout process

### 🍎 Product Categories

- Mac
- iPad

- Vision Pro

### 👤 User Features
- Login & Register
- Account dashboard
- Order history
- Wishlist
- Profile management

### 🌍 Localization
- Azerbaijani

### 🛠️ Admin Dashboard
- Add products
- Edit products
- Delete products

- Category management

### 📩 Support
- Submit support requests
- Upload product issue images
- Describe product problems
- Customer assistance form
